# 🏥 Medical Practice Directory

## 📝 Project Description
This is a simple management system developed in **C Language** to manage medical practice records. It allows for efficient handling of doctor and patient data with a permanent file-based storage system.

## 🚀 Key Features
* **Doctor Management:** Add and list doctors with their specialties.
* **Patient Records:** Register patients and manage their information.
* **Data Persistence:** Automatically saves all data into binary files (`.dat`) for future use.
