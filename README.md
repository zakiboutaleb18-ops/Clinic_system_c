# 🏥 نظام إدارة عيادة طبية (Medical Practice Directory)

## 📝 وصف المشروع
هذا البرنامج عبارة عن تطبيق بسيط بلغة **C** مصمم لإدارة بيانات الأطباء والمرضى. يتميز البرنامج بخاصية **التخزين الدائم**، حيث يتم حفظ كافة البيانات في ملفات ثنائية لضمان بقائها حتى بعد إغلاق البرنامج.

---

## 🚀 المميزات الرئيسية
* **إدارة الأطباء:** إضافة وعرض بيانات الأطباء (رقم التسجيل، التخصص، إلخ).
* **إدارة المرضى:** تسجيل المرضى ومتابعة مواعيدهم.
* **نظام الملفات:** حفظ تلقائي في ملفات `doctors.dat` و `patients.dat`.
